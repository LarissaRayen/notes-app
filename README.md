# Notes App (React Native)

A mobile notes application built with React Native that allows users to securely sign up, log in, and manage personal notes. Users can create, edit, and delete notes, with all data stored and synced using Appwrite as the backend database and authentication service.

---

## Features

- User authentication
- Create new notes
- Edit existing notes
- Delete notes
- Secure user-specific data storage
- Persistent backend using Appwrite

---

## Tech Stack

- **Frontend:** React Native (Expo)
- **Backend / Database:** Appwrite
- **Authentication:** Appwrite Auth
- **Navigation:** Expo Router

---

## Screens / Functionality

- Authentication screens
- Notes list screen
- Create note functionality
- Edit note functionality
- Delete note functionality
